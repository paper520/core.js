## Installation

In a browser:
```html
<script src="./dist/core.js"></script>
```

use:
```js
__.XXX();
```

Using npm:
```shell
$ npm i @yelloxing/core.js
```

Note: add --save if you are using npm < 5.0.0

```js
const __ = require('@yelloxing/core.js');
```

or

```js
import __ from '@yelloxing/core.js';
```
